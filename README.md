🌟 Fun Times Tables! — Learning App for Kids
Made with love for Princess Anya De Alwis 👑❤️ By Mom & Dad
A colorful, interactive web app designed to help young children (ages 5–8) master their times tables through play. Built as a single HTML file that works completely offline — no internet, no installation, no accounts needed. Just open and learn!
What's Inside
6 Fun Game Modes to keep learning fresh and exciting:

📖 Learn — guided practice from ×1 to ×10, one question at a time with instant feedback and encouraging messages. Perfect for beginners.
⚡ Speed Round — a 60-second quiz blitz! Answer as many multiple-choice questions as possible before the timer runs out. Features countdown sounds and streak bonuses.
✏️ Fill in the Blank — type the answer instead of picking from choices. Tests real recall and builds confidence. Also timed at 60 seconds.
🧩 Match Up — pair questions with their correct answers in a tile-matching game. Tracks completion time so kids can try to beat their personal best.
🧗 Mountain Climb — answer 10 questions to climb to the summit. A visual step tracker shows progress with each correct answer.
❤️ Survival — start with 3 lives and see how far you can go! No timer, just accuracy. One wrong answer costs a life.

Key Features

Child-friendly design — large buttons, bright colors, big text, and cartoon emojis make it easy and fun for little hands and eyes.
Works on any device — phones, tablets, laptops, and desktops. Fully touch-friendly and responsive.
Sound effects — cheerful chimes for correct answers, gentle feedback for wrong ones, streak sounds, countdown ticks, level-up fanfares, and a special celebration for perfect scores. All generated in the browser — no audio files needed.
Celebrations & animations — confetti bursts, button animations, and streak fire emojis keep kids motivated.
Streak system — tracks consecutive correct answers with escalating excitement ("🔥 5 in a row!").
Level progression — kids level up from Beginner → Learner → Practiser → Pro → Star Student → Maths Wizard based on total correct answers.
Star ratings — every game ends with a 1–5 star rating and personalized encouraging messages.
Progress tracking — shows total questions answered, accuracy percentage, games played, best score, and best streak.
🔒 Parent dashboard — a password-protected area (default: 1234) where parents can see detailed stats, weak spots (most-missed questions), and full game history.
Flexible table selection — built for the 3× table but easily switchable to any table from 2 to 10 using the selector on the main menu.
